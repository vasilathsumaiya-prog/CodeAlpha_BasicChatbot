# Basic Chatbot

A GUI-based chatbot developed using Python Tkinter.
The chatbot responds to predefined user messages through a simple graphical interface.

---

## Features

- GUI chatbot interface
- Rule-based responses
- Send button interaction
- Multiple chatbot replies
- User-friendly design

---

## Technologies Used

- Python
- Tkinter
- Functions
- If-Else Conditions

---

## Concepts Used

- functions
- if-elif
- input/output
- GUI programming

---

## How to Run

1. Install Python
2. Open VS Code or terminal
3. Run the program using:

python chatbot.py

---

## Project Output

The chatbot window allows users to:
- Enter messages
- Receive bot replies
- Interact through a GUI interface
